
## 📌 Overview

This project was developed as part of the CODEHER 26 hackathon.

The main aim of this project is to provide **career guidance for rural students** who may not have access to proper resources or mentorship. The platform helps students explore different career paths and make informed decisions about their future.

To make the experience engaging, the application also includes a **reward-based system**, where users can earn coins by interacting with the platform.

---

## 🌟 Key Features

* 🎓 Career guidance for students
* 🧭 Information on different career paths
* 🪙 Reward system with coins
* 📱 Simple and user-friendly interface
* 💻 Responsive web design

## 🎯 Problem It Solves

Many rural students lack:

* Access to proper career guidance
* Awareness about different opportunities
* Motivation and engagement in exploring careers

This project aims to bridge that gap in a simple and interactive way.
